Image Processing Program

A program that can resize an image using a seam-carving algorithm using pointers, streams, and abstract data types in C++; it uses its algorithm that carves content-aware resizing of images. This project was made for EECS 280: Programming and Intro Data Structures. I recieved minimal starter code and instructures, implemented the Jpeg implementation without instruction, and recieved an A grade.

Technologies/Skills:

  • C++
  • PPM/JPEG Handling
  • Object-Oriented Programming
  • LibJPEG-turbo
  • GNU Make
  • Input/Output Streams
